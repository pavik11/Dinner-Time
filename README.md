# Dinner-Time

CS50 Week 0: Dinner Time

Instructions:
1. Press the egg to hatch the chick and start the game.
2. But BE AWARE, a hungry snake might be lurking as soon as the chick is hatched.
3. Move the chick around with the arrow keys to avoid the hungry snake and become dinner.
4. You have 3 lives to wander around the screen and stay alive for as long as possible and the levels get harder.
5. Have Fun!
